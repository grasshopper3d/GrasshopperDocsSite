Filter UI Elements

Home User Interface Design addonsHuman UIUI MainFilter

This component allows you to select UI elements from a window by name, to let you listen for their values or set their properties dynamically.

Inputs

NameIDDescriptionType
ElementsEAll Controls and other elements belonging to the windowGeneric Data
Name Filter(s)FThe filter(s) for the elements you want to listen for.Text

Outputs

NameIDDescriptionType
Filtered ElementsFEThe selected UI elements.Generic Data
