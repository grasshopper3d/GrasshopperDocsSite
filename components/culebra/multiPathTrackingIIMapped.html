Multi Path Tracking II Mapped

Component Index Culebra03 | BehaviorsTT

MultiShape Path Following Algorithm capable of spawning children with image color sampling override for any path attributes and remaping of color values - see example files

Inputs

NameIDDescriptionType
PolylinesPInput a list of polylines you want to follow, POLYLINE RESOLUTION IS IMPORTANT, KEEP AS LOW AS POSSIBLECurve
Polyline ThresholdPTInput the distance threshold enabling agents to see shapesNumber
Projection DistancePDInput the projection distance of point ahead on the path to seekNumber
Polyline RadiusPRInput the radius of the shapesNumber
Trigger SpawnTSInput value specifying if creeper is now allowed to spawn any children objects storedBoolean
Max ChildrenMCInput value specifying the maximum number of children each creeper can haveInteger
Colored MeshCMInput a color mesh to drive the path parametersMesh
Map ThresholdMTInput value specifying if you want the path threshold value to be color drivenBoolean
Map ProjectionMPInput value specifying if you want the projection value to be color drivenBoolean
Map RadiusMRInput value specifying if you want the path radius value to be color drivenBoolean

Outputs

NameIDDescriptionType
Tracking BehaviorTBThe tracking behavior data structureGeneric Data
