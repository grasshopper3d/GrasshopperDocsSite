UDP Receiver

Component Index gHowlUDP>UDP<

Allows to receive data on the network

Inputs

NameIDDescriptionType
Address@The local IP address to listen at. If empty, the loopback address is usedString
PortPThe port number to listen atInteger
Pattern#The pattern:\r\n0 = Parse received bytes as text\r\n999 = OSC Message\r\nInteger

Outputs

NameIDDescriptionType
InfoINetwork informationString
DataDThe string which was just receivedString
