Render Curve to Screen

Displays a Curve object in a fixed position on screen

Inputs

NameIDDescriptionType
CurveCrvCurve to display to screen. Make sure the curve is positioned at (or near) the world origin.Curve
LocationLThe location to display the text. If \"Scale With Viewport\" is unchecked, specify points in pixel coordinates - \n 100,100,0 will be 100 pixels right and down from the upper-left corner of the viewport.Point
Scale FactorSA multiplier on the scale. Use this to adjust the size of the curve in the display.Number
ColorCThe color to display the Curve.Colour
ThicknessTThe lineweight to display the curve.Integer
DepthDA value between 0-1 that controls the Z-depth of the curve display. \nA high number will appear in front of most objects - low will go behind.\nHowever, values exactly equal to 0 or 1 should be avoided - stay slightly within the \nlimits to avoid artifactsNumber
FixedFIf true, objects remain fixed, oriented to the screen plane - if false, objects rotate when the viewport does.Boolean
