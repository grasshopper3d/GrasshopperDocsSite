Explode Blocks

Explodes a set of Instance Objects into their proper geometry

Inputs

NameIDDescriptionType
Block ObjectsblockData in the form of Rhino.DocObjects.InstanceObjectGeneric
Block Name FilterBNFAn optional filter for the name of the block. Only blocks matching the filter (w/ wildcards like *, ?) will be processed/exploded.Text

Outputs

NameIDDescriptionType
GeometrygeoBlock GeometryGeneric
AttributesattObject AttributesGeneric
Block NamenameThe Block definition nameText
BoundingBoxbboxThe Bounding box around the instance definitionBox
Block TransformxformThe transform of this block instanceTransform
Insertion PlaneplnThe insertion point of this block instance.Plane
Instance CountctThe number of instances of this block present in the Rhino document.Integer
