SpaceGassSolver

Design Steel

Home Structural Analysis addonsGeomGym SpacegassSpaceGassSolverggD

This component will execute Structural Analysis solution and Steel design process. Advisable to have only one design component per model and manage result extraction using list access in Grasshopper. Ensure that SpaceGass is not open prior to running the Solver.
