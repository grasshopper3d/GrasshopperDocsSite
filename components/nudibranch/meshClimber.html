Mesh Climber

Component Index NudibranchScramblersMeshClimber

Generates the gradient descent or ascent paths on a Mesh object for any provided point list.

Inputs

NameIDDescriptionType
MeshTerrainMMesh Surface terrainMesh
PointsPPoint List to affectPoint
ForceFGlobal (gravity or reverse gravity) force for all the particlesNumber
IncrementalInGlobal angle for all the particles resulting in an incremental descent or ascent(radians)Number
IterationsItNumber of iterationsInteger

Outputs

NameIDDescriptionType
NewPointsPThe generated point list per iterationPoint
NewVectorsVThe generated vector list per iterationVector
PathsCThe generated path from each point's route on the surfaceCurve
