Sample KDTree

Component Index TarsierSearchingKDSample

Builds and/or samples a KDTree representation of a point cloud.

Inputs

NameIDDescriptionType
KD-TreeKThe KD-Tree to sample.Goo
PointsPThe points to sample.Point
RangeRThe search range.Number
Maximum Points ReturnedNThe maximum number of points to be returned for each query point.Integer

Outputs

NameIDDescriptionType
IndicesIThe indices of the neighbouring points.Integer
