Wind Thresholds

Compute the Lawson criteria for a CFD case.

Inputs

NameIDDescriptionType
InputInputInput from the previous Compute ComponentText
EPW FileEPW FilePath to where the EPW file is located.Text
Probe NamesProbesGive names to each branch of in the points tree. The names can later be used to identify the points.Text
ThresholdsThresholdsThresholds for different wind comfort categories. Input should be a list JSON formatted strings, with the fields: "field" and "value", respectively describing the category name and threshold value. The default values corresponds to the Lawson 2001 CriteriaText
CPUsCPUsCPUs to use. Valid choices are: 1, 2, 4, 8, 16, 18, 24, 36, 48, 64, 72, 96Integer
DependentOnDependentOnBy default the probe task is dependent on a wind tunnel task or a task running simpleFoam. If you want it to be dependent on another task. Please supply the name of that task here.Text
CreateCreateWhether to create a new Wind Threshold task, if one doesn't existBoolean
