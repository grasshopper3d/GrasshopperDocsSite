Connection From Direction

Component Index Wasp1 | ElementsConnection From Direction

Create a connection on a given part geometry from center and X-asix direction
(Z direction is determined based on the normals of the component geometry)
-
Provided by Wasp 0.2

Inputs

NameIDDescriptionType
GEOGEOGeometry of the part to which the connection belongsGoo
CENCENOrigin of the connection planeGoo
UPUPLine idetifying the X-axis of the connection plane - determines the plane orientation to mantain when transformingGoo
TTOPTIONAL // Connection type (to be used with Rule Generator component)Goo

Outputs

NameIDDescriptionType
CONNCONNConnection objectGoo
PLN_OUTPLN_OUTPlane for each connection object (for debugging)Goo
