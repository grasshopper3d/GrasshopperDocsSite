Quadri Design Task Selector

Gets valid Design Tasks from an open Quadri model.
Release notes v0.0.22 using TaskConnector
* Added output node to Select Quadri Feature component, separating features and tasks.
* Added refresh toggle on Select Quadri Task to facilitate workflow where tasks needs to be updated from outside the script.
* Added new component Quadri Model Info which presents info available about the current active model.
