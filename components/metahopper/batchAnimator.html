Batch Animator

Saves a screenshot of every state of the model in the batch to a folder.

Inputs

NameIDDescriptionType
PathpathThe path output of the BatchDriver componentText
File ExtensionextThe file extension to save the animation frames, e.g. jpg, png, tif, gif, bmpText
Viewport NameviewThe name of the viewport to save. Defaults to the active viewport.Text
Image WidthxDimThe width of the image to save. Defaults to the active viewport width.Integer
Image HeightyDimThe height of the image to save. Defaults to the active viewport height.Integer
