Polygon Topology Edge Filter

Inputs

NameIDDescriptionType
Edge listEOrdered list of edgesLine
Edge-Loop structureELOrdered structure listing the polylines adjacent to each edgeInteger
Valency filterVFilter edges with the specified number of adjacent polylinesInteger

Outputs

NameIDDescriptionType
List of edge IDsIList of edge indices matching the valency criteriaText
List of edgesEList of edges matching the valency criteriaLine
