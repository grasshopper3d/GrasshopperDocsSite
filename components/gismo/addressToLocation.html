Address To Location

Use this component to find coordinates of a specific location using an address.
It uses Nominatim API
-
Provided by Gismo 0.0.3

Inputs

NameIDDescriptionType
Address_addressA string representing the address for which location (latitude and longitude coordinates) is suppose to be found.Goo
Openweb_openwebIf True, open the browser with nominatim (False by default)Goo

Outputs

NameIDDescriptionType
OutreadMe!explained result of the queryString
Locationlocation[address, latitude, longitude coordinates) of the input.Goo
WeblinkweblinkScript variable AddressToLocationGoo
