OSM Keys

Use this component to generate a list of keys for the "requiredKeys_" input of "OSM shapes" component.
A list of keys depends on the chosen OSM object name (_OSMobjectName). Use "OSM Objects" dropdown list to generate the appropriate _OSMobjectName.
-
Provided by Gismo 0.0.3

Inputs

NameIDDescriptionType
OSMobject Name_OSMobjectNameOSM object name. Use "OSM Objects" dropdown list to generate it.Goo

Outputs

OutreadMe!...String
Required KeysrequiredKeysA list of keys for the chosen OSM object name (_OSMobjectName). Supply it to the "requiredKeys_" input of "OSM shapes" component.Goo
