Surface Environment

Component Index QueleaEnvironmentsSurface Environment

A Surface Environment upon which the Quelea will map their position for the calculation of Forces.

Inputs

NameIDDescriptionType
SurfaceSAn untrimmed Surface.Surface
Wrap?WIf true, quelea will wrap around to other side of environment bounds.Boolean

Outputs

NameIDDescriptionType
EnvironmentEnA geometrical object representing the environment the Quelea will interact with.Goo
2D Reference SurfaceRSThe surface mapped to a 2D plane using the input surface's u and v dimensions.Surface
