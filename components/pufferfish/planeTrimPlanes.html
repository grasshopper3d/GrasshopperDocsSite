Plane Trim Planes

Component Index PufferfishPlanePlnTrimPlns

Trim (remove) the parts of a plane set on one side of a plane.

Inputs

NameIDDescriptionType
PlanesPlSet of planes to trimPlane
FlipFFlip the trim directionBoolean

Outputs

NameIDDescriptionType
TrimmedTResulting trimmed plane setPlane
Kept IndiciesiIndices from the input plane set of the planes that remained after the trim operationInteger
IntersectedITrue if the plane trimmed the plane set, False if the plane did not trim the plane setBoolean
