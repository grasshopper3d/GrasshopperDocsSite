Construct Drawing

Constructs a Drawing from a list of Shapes

Inputs

NameIDDescriptionType
Shapes / GeometrySA list of Graphic Plus Shapes, or Curves, Breps, MeshesGeneric Data
BoundaryBAn optional frame for the drawing. If blank, the shapes bounding box will be usedRectangle
WidthWThe width of the output drawingNumber
HeightHThe height of the output drawingNumber
ColorCAn optional background colorColour
UnitsUAn optional unit scale for the drawingInteger

Outputs

NameIDDescriptionType
DrawingDA Graphic Plus Drawing ObjectGeneric Data
BoundaryBThe bounding rectangleRectangle
