Multi Mesh Patch

Component Index HeronUtilitiesMMPatch

Multithreaded creation of mesh patches from planar polylines. The first polyine in a branch will be considered the outer boundary, any others will be considered holes and should be completely within the outer boundary.

Inputs

NameIDDescriptionType
PolylinespolylinesPolylines from which to generate mesh patches. The first polyine in a branch will be considered the outer boundary, any others will be considered holes and should be completely within the outer boundary.Curve
Extrude HeightextrudeHeight to extrude mesh patches.Number

Outputs

NameIDDescriptionType
Mesh PatchesmeshesMeshes resulting from mesh patches.Mesh
