Import Vector Lite

This is a basic version of the Import Vector component allowing the import of GIS data, including SHP, GeoJSON, OSM, KML, MVT and GDB folders. The data will be imported in the source's units and spatial reference system, Heron's SetSRS and Rhino's EarthAnchorPoint will have no effect.
