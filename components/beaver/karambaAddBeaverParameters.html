Karamba add beaver parameters

Component Index BeaverExternalB2KBeams

Adds Beaver parameters to a Karamba Beam Element for performing ULS and SLS checks in Beaver

Inputs

NameIDDescriptionType
BeamBeamKaramba Beam element to be ModifiedElement
Span LineSpanLineSpan polyline or polyline representing the full undeformed length of the beam. This is used for defining local considerations of SLS. If no line is provided, SLS will consider a global displacementCurve
Buckling Length YBklLenYEffective buckling length of element in local Y-direction if > 0. (for beams consider EC5 table 6.1)Number
Buckling Length ZBklLenZEffective buckling length of element in local Z - direction if > 0. (for beams consider EC5 table 6.1)Number
Buckling Length LTBklLenLTEffective buckling length of element in lateral torsional buckling if > 0. (for beams consider EC5 table 6.1)Number
CantileverCantileverBoolean parameter describing whether the beam is cantilevered. By default is set to FalseBoolean
Service ClassSCService Class from 1 to 3 regarding the timber element. By default is set to 2Integer
PrecamberPcamberPrecamber of beam elements in meters. Default is set to 0Number
LocalLocalBoolean parameter describing whether the beam should be calculated using global or local SLS analysis. By default is set to FalseBoolean

Outputs

NameIDDescriptionType
BeamBeamKaramba Beam element to be ModifiedElement
