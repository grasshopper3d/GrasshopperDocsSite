Weave MGraphs

Component Index Ivy9|Other ToolsWeave

Weave two MeshGraph segmentations

Inputs

NameIDDescriptionType
MeshGraph1G1The first set of meshgraph segments. All segments must be tree graphs and all segments must come from the same original MeshGraphAny
MeshGraph2G2The second set of meshgraph segments. All segments must be tree graphs and all segments must come from the same original MeshGraphAny
OffsetoThe maximum deviation of the weave from the original meshNumber

Outputs

NameIDDescriptionType
MeshGraph1G1The first set of weaved meshgraph segments.Any
MeshGraph2G2The second set of weaved meshgraph segments.Any
First meshesM1The first set as weaved meshesMesh
Second meshesM2The second set as weaved meshesMesh
