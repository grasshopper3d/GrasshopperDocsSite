Custom Face Weight

Component Index Ivy2|Weight GraphcFaceWeight

Assign a custom weight to graph nodes (mesh Faces). Values are normalized in the 0..1 interval

Inputs

NameIDDescriptionType
MeshGraphGThe MGraph object that will receive the new weight valuesAny
WeightsWValues to be added to the mesh faces as weightNumber
