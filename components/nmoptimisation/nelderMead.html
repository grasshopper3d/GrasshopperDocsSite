NelderMead

Component Index NM OptimisationOptimisationNM Opti

Nelder-Mead optimisation is a local search method good for when computation is slightly slower and a reasonable starting point is available

Inputs

NameIDDescriptionType
VariablesxA number of sliders which represent optimisation variablesNumber
FitnessfA value which represents the fitness of the solution. This value will be minimised or maximised during the optimisation proceedureNumber
Constraint utilisationgg < 1.0 - feasible solution; g > 1.0 infeasible solution (constraint not met)Number
