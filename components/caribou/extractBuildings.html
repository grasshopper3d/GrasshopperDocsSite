Extract Buildings

Home Maps & GIS addonsCaribouParseBuildings

Load and parse node (e.g. point) data from an OSM file based on its metadata

Inputs

NameIDDescriptionType
OSM File PathFPThe path to XML file(s) downloaded from Open Street mapText
OSM FeaturesOFA list of features and subfeatures to extract from the OSM file, in a 'key=value' format separated by newlines or commasText
Output HeightedOH?If true, only outputs buildings with height data. If false, only outputs buildings without height data.Boolean

Outputs

NameIDDescriptionType
BuildingsBBuildings as extrusions from associated way geometriesBrep
TagsTThe metadata attached to each particular nodeText
ReportRThe name, description, and number of items found of each specified featureText
BoundsBThe boundary extends of the OSM file(s)Rectangle
