Colibri Image Setting

Component Index ColibriColibriSet Img

Set the parameters for view captures from each iteration.
You can specify which viewport[s] to capture, and the resolution of the image.
NOTE: It is recommended to use the Colibri Preview components to assure the viewport has finished updating before the image is captured.

Inputs

NameIDDescriptionType
ImageSaveAsNameSOptional input for overwriting the default image name. Don't do anything here if you don't know how to generate a dynamic name!Text
RhinoViewNamesVOptional input for the Rhino viewport name which you would like to take a snapshot of. Acceptable inputs include "Perspective", "Top", "Bottom", etc or any view name that you have already saved within the Rhino file (note that you do not need to input quotations). If no text is input here, the default will be an image of the active viewport - the last viewport in which you navigated.Text
WidthWImage width in pixels.Integer
HeightHImage height in pixels.Integer

Outputs

NameIDDescriptionType
ImgSettingISColibri's image setting object. Feed this into the Colibri aggregator downstream.Generic Data
