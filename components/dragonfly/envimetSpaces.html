Envimet Spaces

Component Index Dragonfly3 | EnvimetDFEnvimetSpaces

Use this component to generate ENVI-Met v4.4.4 3D geometry models.
Analyze parametric models with ENVI - Met!
Save the model in the ENVI_MET Workspace, create a simulation file with "DF Envimet Config" and run the simulation.

Inputs

NameIDDescriptionType
envimetFolder_envimetFolderThe folder into which you would like to write the envimet model. This should be a complete file path to the folder.String
envimetLocation_envimetLocationLocation data which comes from "Dragonfly Envimet Location" component.Goo
envimentGrid_envimentGridConnect the output of "Dragonfly Envimet Grid".Goo
nestingGridnestingGrid_Connect the output of "Dragonfly Envimet Nesting Grid".Goo
envimentObjects_envimentObjects_Connect envimet buildings, trees, soils, source, terrain here.Goo
fileNamefileName_The file name that you would like the envimet model to be saved as. Default name is "DragonflyEnvimet".String
runIt_runItSet to "True" to run the component and generate the envimet model.Boolean
viewGridviewGrid_Set to "True" to run the view grid.Boolean

Outputs

NameIDDescriptionType
ModelInfoModelInfoInformation about Model grid.String
XYGridXYGridXY points.Point
XZGridXZGridXZ points.Point
YZGridYZGridYZ points.Point
INXfileAddressINXfileAddressThe file path of the inx result file that has been generated on your machine.String
