Envimet Run Simulation

Use this component to run directly simulation by Grasshopper.

Inputs

NameIDDescriptionType
envimetFolder_envimetFolderEnvimet project folder which comes from "DF Envimet Manage Workspace".String
SIMXfileAddress_SIMXfileAddressConnect the output of DF Envimet Config.String
type_type_False = 32-bit or True = 64-bit. Default is 64-bit.Boolean
runIt_runItSet runIt to "True" to run model.Boolean
