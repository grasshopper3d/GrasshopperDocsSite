Tekla Live Link

ADDON. Version 1.6. Released on 2020.1.6. Provides 0 components. Created by Trimble Solutions Corporation. Features 0 video tutorials.
The Grasshopper-Tekla live link is a set of Grasshopper components that can create and interact with objects live in Tekla Structures. The link enables structural engineers who create free-form roofs and facades to efficiently support the architectural vision in early conceptual design phase. Other practical applications of the link include: Creation of building stick frames to drive analysis and Building Information Models, Model beams and slabs with unusual curvature, Generation of detailed steel sections parametrically. In addition, the link can be used to create complex geometry for bridges and other structures.
