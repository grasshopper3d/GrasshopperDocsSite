Dodo

ADDON. Version 0.3. Released on 2019-Feb-23. Provides 44 components. Created by Lorenzo Greco. Features 1 video tutorials.
Dodo is a collection of tools for machine learning, optimization, and geometry manipulation. Regarding AI, it features neural networks, gradient descent, stochastic gradient descent and swarm optimization. Dodo has tools for scalar, vector and tensor fields manipulation which can be visualized using isosurfaces. Some components are implementations of papers, you can find more info and reference in their html description.
