GhExcel

ADDON. Version 1.2. Released on 2013-Jun-01. Provides 3 components. Created by Xiaoming. Features 0 video tutorials.
GhExcel is a plug-in for Grasshopper to exchange data with Excel. Multiple sheets are supported with ZUI(Zoomable User Interface).
