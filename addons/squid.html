Squid

ADDON. Version 0.110. Released on 2015-Jan-22. Provides 25 components. Created by Mateusz Zwierzycki. Features 0 video tutorials.
This is the first release of my plug-in to draw bitmaps with Grasshopper. This utility for Grasshopper exposes a bunch of components in the Squid tab.
