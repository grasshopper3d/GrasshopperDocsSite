Groundhog

ADDON. Version 0.13.0. Released on 2023-Dec. Provides 25 components. Created by Philip Belesky. Features 0 video tutorials.
Groundhog is a Grasshopper plugin, a set of reference models, and wiki exploring the applications of computational design in landscape architecture.
