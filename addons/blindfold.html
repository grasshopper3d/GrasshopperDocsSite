Blindfold

ADDON. Version 1.0. Released on 2018-Jun-12. Provides 1 components. Created by Ekim Royrp. Features 0 video tutorials.
Blindfold is a single component which can hide or show geometry in Rhino that is referenced into the Grasshopper document. Blinfold has three modes and an icon which changes depending on the mode it is set to: Off, Hide, or Show. Blindfold by default detects and hides/shows all geometry in Rhino that is referenced into any Grasshopper parameters or component inputs, optionally it can limit the hide/show operation to only the Rhino referenced geometries plugged into or set in the Limited input. When new Rhino geometry is referenced you can refresh the hide/show operation by clicking a button plugged into the Refresh input. Blindfold makes it easy to manage the visibility of your Rhino geometry referenced in Grasshopper so you can quickly avoid those annoying overlap display issues between Rhino and Grasshopper previews.
