Nemo

ADDON. Version 1.1 Beta. Released on 2019-Oct-16. Provides 25 components. Created by VENOT. Features 0 video tutorials.
Naval Efficiency and Marine Optimization. Nemo is a Grasshopper plugin for Ship Design. Nemo Analysis, for automatic analysis of ship geometry (licensed). Nemo Designer, which includes several design tools (free) Nemo Analysis can perform hydrostatics computation, silhouette extraction, meshing generation and resistance prediction right inside Rhino. Sectioning hull into stations - buttocks - waterlines, generating an offset table, or creating cross-sections curves are also embed features. Interoperability is allowed with external software like NavCad (HydroComp), Mars 2000 (Bureau Veritas) or Excel (Microsoft). Nemo Designer includes specific tools to create NACA sections and appendages such as hydrofoils, keels (with or without bulb), or even blade propellers.
