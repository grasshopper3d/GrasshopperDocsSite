Centipede

ADDON. Version 0.7b. Released on 2012-Oct-03. Provides 2 components. Created by Carson Smuts. Features 1 video tutorials.
Allows the user to keyframe the animation of multiple objects in Grasshopper along a single slider. New version 0.7b allows you to export animation to 3D Studio Max for rendering.
