Alpaca4d

ADDON. Version 0.7.1. Released on 19-Nov-2023. Provides 64 components. Created by Marco Pellegrino. Features 0 video tutorials.
Alpaca4d is a Grasshopper plugin which has been developed on top of OpenSees. It lets you analyse beam, shell and brick elements through Static, Modal and Ground Motion Analysis. Alpaca4d facilitates the interaction with OpenSees making it possible to model complex geometries within your parametric workflow and reducing the time spent in the modelling. The main idea is to provide an efficient and easy way to use OpenSees without writing any line of code.
