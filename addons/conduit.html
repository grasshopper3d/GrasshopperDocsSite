Conduit

ADDON. Version v.2018.11.19. Released on 2018-Nov-12. Provides 0 components. Created by Nathan Miller. Features 0 video tutorials.
Conduit is an open source plug-in authored by Proving Ground. Conduit enables designers to create custom data visualizations, dashboards, and heads up displays (HUDs) that update with their computational design tools and parametric models.
