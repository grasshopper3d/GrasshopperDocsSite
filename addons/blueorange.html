BlueOrange

ADDON. Version 1.0.15. Released on 01-Oct-2023. Provides 126 components. Created by wucl12. Features 0 video tutorials.
BlueOrange is a collection of useful utilities for Grasshopper. It provides rectangular geometries, block transforms, and and quick generation tools used for quick design concept modeling stage. Other miscellaneous interesting components like MIDI devices reader (For reading piano signal) or Converger for calculating a singular input-output converging process.
