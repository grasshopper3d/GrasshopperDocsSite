Mouse

ADDON. Version 1.0.0.0. Released on 2018-Sep-07. Provides 2 components. Created by Arash Nader. Features 0 video tutorials.
The Mouse is a pathfinding plugin for Grasshopper based on Theta* algorithm by Kenny Daniel, Alex Nash, and Sven Koenig. It provides the possibility to find optimize and realistic looking routes between each pair of points in Grasshopper.
