Hummingbird

ADDON. Version 1.0.0.6. Released on 2019-May-03. Provides 17 components. Created by Mario Guttman. Features 1 video tutorials.
Hummingbird is a set of Grasshopper components that facilitate the creation of Revit native geometry. This process exports basic geometric properties and parameter data to CSV text files which are used to create Revit BIM geometry. The tool also supports importing Revit geometry into Rhino, making a bi-directional workflow possible.
